# Dinner Invitation & RSVP Tracker

A lightweight **PHP + MySQL web application** that allows members to submit dinner invitations and manage guest RSVP status (**Confirmed / Not Confirmed**) on a **public page with no login required**.

This project was designed for simplicity, clarity, and team collaboration, making it ideal for small events, class dinners, or group gatherings.
